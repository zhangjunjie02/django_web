3.过滤器

  • 原值：{{name}}
  • 大写：{{name|upper}}
  • 小写：{{name|lower}}
  • {{time}}
  • {{time|date:"Y-m-d H:i:s"}}
  • {{a}}
  • {{a|join:"#"}}

4.注释

{#
  • 原值：{{name}}
    • #} {% comment %}
  • 大写：{{name|upper}}
  • 小写：{{name|lower}}
    • {% endcomment %}

    5.运算

    • 两个数：{{m1}}:{{m2}}
    • 求和：{{m1|add:m2}}
    • 求差：{{m1|add:-20}}
    • 相乘：{% widthratio m1 1 m2 %}
    • 相除：{% widthratio m1 m2 1 %}

    6.自定义标签和过滤器

    {% load pagetag %} {{name|myupper}} {% jian m1 m2 %}
